# Library Management System

## About the Project

This is a **library management software** developed using **React.js** on the front-end and **pure PHP** on the back-end.

Communication between the interface and the system is handled through a **simple and clean REST API in JSON format**, with **Axios** being used to perform HTTP requests.

The software allows users to create and manage **multiple libraries**, making it possible to organize books, readers, loans, and collaborators in a centralized environment.

Users can invite coworkers through an invitation system, allowing collaboration within the same library environment.

This project was developed with the goal of **learning and improving skills in pure PHP API development and React.js interface development**.

## Features

* Creation and management of **multiple libraries**
* **Invitation system** for adding users and coworkers
* User collaboration within libraries
* Create, edit, and delete **books**
* Create, edit, and delete **readers**
* Create, edit, and manage **book loans**
* Simple API communication using **JSON**
* HTTP requests handled with **Axios**

## Technologies Used

* **Front-end:** React.js
* **Back-end:** Pure PHP
* **Communication Format:** JSON
* **HTTP Client:** Axios

## Purpose

The main purpose of this project was to deepen knowledge in:

* **Pure PHP back-end development**
* **Simple API development**
* **React.js front-end development**
* **Client-server communication**
* **System architecture and CRUD operations**

The project focuses on both practical learning and software development, simulating features commonly found in real-world library management systems.
